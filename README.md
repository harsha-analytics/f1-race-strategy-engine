#F1 Race Strategy Decision Engine

This project is a Python-based decision engine that simulates Formula 1 race strategy decisions.

## Features
- Computes tyre degradation rates
- Compares performance between our car and rival drivers
- Calculates per-lap advantage
- Estimates recovery laps after pit stop
- Determines undercut viability
- Recommends optimal race action (PIT / STAY OUT)
- Provides confidence level for decisions

## Streamlit Dashboard
An interactive dashboard allows users to input lap times and simulate strategy outcomes.

## How it works
The engine processes lap time data, calculates degradation trends, and evaluates whether a pit stop will provide a competitive advantage.
